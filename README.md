# Sparse Coding (Olshausen & Field, 1996) Model

# Requirement
- `Python >= 3.5`
- `numpy`, `matplotlib`, `scipy`, `opencv`, `tqdm`

## Usage
- Run `train.py` or `predictive-coding.ipynb` (written in Japanese).
- `preprocessing.py`, `ica.py`, `input_test.py` are test files.
